# ASIC-Memory-RTL-to-GDSII
Complete RTL-to-GDSII implementation of an ASIC Memory design using Cadence Genus and Innovus, covering synthesis, floorplanning, CTS, routing, timing analysis, and physical design.  
Memory RTL-to-GDSII
Overview

This repository presents the complete RTL-to-GDSII implementation of a Memory design using the Cadence digital design flow. The project covers the major stages of ASIC implementation, starting from RTL synthesis and progressing through physical design to the final layout generation. It demonstrates practical knowledge of digital backend design and industry-standard EDA tools.

Tools Used
Cadence Genus
Cadence Innovus
Verilog HDL
TCL
SDC
Design Flow

The implementation includes the following stages:

RTL Design
Logic Synthesis
Timing Constraint Definition
Floorplanning
Power Planning
Standard Cell Placement
Clock Tree Synthesis (CTS)
Routing
Static Timing Analysis (STA)
Timing Optimization and Closure
GDSII Generation
